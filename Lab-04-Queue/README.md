# Lab 4 - Queue Implementation in Java

## Objective
Implement Queue data structure in Java without using Collection Framework.

---

## Features
- Enqueue Operation
- Dequeue Operation
- Display Queue
- Printer Queue Simulation

---

## Queue Principle

Queue follows:

FIFO (First In First Out)

Example:

10 → 20 → 30

Dequeue removes:
10 first

---

## Algorithms

### Enqueue Algorithm
1. Check queue overflow
2. Increment rear
3. Insert element

### Dequeue Algorithm
1. Check queue underflow
2. Remove front element
3. Increment front

### Printer Queue Algorithm
1. Insert print jobs
2. Process jobs one by one
