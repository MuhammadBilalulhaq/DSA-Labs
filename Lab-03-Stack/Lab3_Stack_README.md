# Lab 3 – Stack Implementation (Java Without Collection Framework)

## Objective

Implement Stack data structure using arrays in Java without using Collection Framework.

---

## Operations

* Push
* Pop
* Peek
* Display

---

## Problem 1

Push 5 numbers into stack, pop 2 numbers, and display remaining elements.

---

## Problem 2 – Reverse String Using Stack

### Algorithm

1. Push all characters of string into stack
2. Pop characters one by one
3. Display reversed string

---

## Concept Used

* Array-based implementation
* LIFO (Last In First Out)
* Stack Overflow handling
* Stack Underflow handling

---

## Rules

* No Java Collection Framework allowed
* Implementation must be from scratch
