# Lab 02 - Linked List (DSA)

This lab contains implementation of core Linked List operations in Java.

---

## 📌 Core Operations Covered

---

## ➤ Traversal Operations

- Print Linked List
- Find length of Linked List
- Search an element in Linked List

---

## ➤ Insertion Operations

- Insert at beginning
- Insert at end
- Insert at specific position
- Insert after a given node
- Insert before a given node

---

## ➤ Deletion Operations

- Delete from beginning
- Delete from end
- Delete by position
- Delete by value
- Delete entire linked list

---

## 💡 Concepts Used

- Singly Linked List
- Node structure
- Pointers (references in Java)
- Dynamic memory allocation concept
- Basic algorithm design

---

## 🧠 Time Complexity

- Traversal: **O(n)**
- Insertion at beginning: **O(1)**
- Insertion at end: **O(n)** (if no tail pointer)
- Insertion at position: **O(n)**
- Deletion at beginning: **O(1)**
- Deletion at end: **O(n)**
- Deletion by value/position: **O(n)**

---

## 🛠 Language Used
- Java
